# Getting Started
Make sure you have node installed
1. Clone repository
2. Use json_gen.js file to run:
```
$ json-server json_gen.js
```
3. Open browser
```
localhost:3000
```
4. Try different routes (refer to [JSON-server documentation](https://github.com/typicode/json-server))
